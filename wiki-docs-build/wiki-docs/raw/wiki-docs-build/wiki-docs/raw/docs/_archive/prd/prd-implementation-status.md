## 6. 우선순위별 개발 계획

   - 기본 레이아웃

   - API 연동

### Phase 2 (중요) - 6주

**목표**: 핵심 분석 기능

1. **LLM 통합**
   - Gemini API 연동

   - 요약 프롬프트 최적화

2. **토픽 모델링**
   - BERTopic 구현
   - 시계열 추이 분석

3. **조기경보 시스템**
   - 임계치 설정 UI

### Phase 3 (개선) - 4주

**목표**: 고급 기능 및 최적화

1. **이벤트 분석**

2. **성능 최적화**
   - 캐싱 (Redis)
